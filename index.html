Sync from Apple Health
Heads up: browsers can't talk to Apple Health directly. Build a 30-second iOS Shortcut that reads from Health and copies a JSON line — then paste it below. The dashboard will update.
How to make the Shortcut →
  1. Open Shortcuts → New → name it "Sync Dashboard"
  2. Add Get Health Sample for Sleep Analysis (Today), Steps (Today), and Active Energy (Today)
  3. Add Text action with JSON like {"sleep":7.5,"steps":8431,"energy":420} using each value
  4. Add Copy to Clipboard
  5. Run it daily, then paste here. Or pin it to your Lock Screen for one-tap sync.
